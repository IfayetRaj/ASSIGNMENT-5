Question-1: What are the different ways to select an element in the DOM?

getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerText means the text inside a tag like h tag, p tag, innerHTML means the whole html inside a tag like div tag. textContent similar to innerText but faster and does not account for styling like innerText.

Question-3: What is event delegation in the DOM?

Event delegation is a way in JavaScript where you attach a single event listener to a parent element to manage events on dynamically added or multiple child elements like in ol, ul, div.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event is triggered on a child element, it is propagated through the parent elements until it reaches the top of the DOM tree.
